# Contract: 외부 랜덤 고양이 API (cataas)

애플리케이션이 **소비**하는 외부 오픈 API 계약(우리가 정의하지 않음 — 관측/가정 기록).

## 엔드포인트

```
GET https://cataas.com/cat/cute
```

- 사용자가 지정한 엔드포인트(spec FR-002 / Assumptions).
- 인증 없음(공개 API 가정). 별도 API 키·헤더 불필요.
- 캐시버스팅을 위해 임의 쿼리 파라미터를 덧붙일 수 있음: `?t=<mount 고유 토큰>` (FR-008).
  선택적으로 `?width=` 등 cataas 파라미터도 허용되나 본 기능에선 필수 아님.

## 응답

| 항목 | 값/가정 |
|---|---|
| 성공(2xx) | 이미지 바이트(`Content-Type: image/*`, 예: `image/jpeg`). "cute" 태그의 랜덤 고양이 1장. |
| 반환 특성 | **요청마다 랜덤** — 동일 URL이라도 새 요청이면 다른 이미지가 올 수 있음. |
| 크기 | 임의(고정 아님). 앱은 `object-fit: cover`로 고정 높이 컨테이너에 맞춤. |

## 소비 방식 (우리 측)

- 표준 `<img src>`로 로드(별도 `fetch`/JSON 파싱 없음). `next.config` 원격 이미지 설정 불필요
  (research §1).
- **성공 신호**: `<img>` `load` 이벤트 → 커버 표시.
- **실패 신호**: `<img>` `error` 이벤트(네트워크 오류/비200/차단 등) → 커버 collapse.

## 실패 모드 & 처리

| 실패 | 관측 | 처리 |
|---|---|---|
| 네트워크 오류·타임아웃(브라우저) | `error` 이벤트 | collapse(FR-005). 인위적 타임아웃은 두지 않음(FR-009) |
| 비-이미지/오류 응답 | `error` 이벤트 | 동일하게 collapse |
| 매우 느리지만 진행 중 | 이벤트 미발생 | 스켈레톤 유지, 편집은 즉시 가능(FR-009, SC-004) |
| 오프라인 | `error` 이벤트 | collapse |

## 우리가 하지 않는 것 (범위 밖)

- 응답 캐싱·재시도·백오프 없음(YAGNI, FR-009).
- 이미지 저장·게시글 연결 없음(FR-008: 저장·고정하지 않음).
- 서버 사이드 프록시/최적화 없음.
